# Grindpoint PWA Icons

## Required Icon Sizes
Generate PNG icons at these sizes from `icon.svg`:
- icon-72.png (72×72)
- icon-96.png (96×96)
- icon-128.png (128×128)
- icon-144.png (144×144)
- icon-152.png (152×152)
- icon-192.png (192×192)
- icon-384.png (384×384)
- icon-512.png (512×512)

## Free Tools to Generate Icons
1. **https://realfavicongenerator.net** — Upload icon.svg, download all sizes
2. **https://favicon.io** — SVG to PNG converter
3. **https://www.pwabuilder.com** — Full PWA icon generator
